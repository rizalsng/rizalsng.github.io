# rizalsng.github.io

Personal website of **Fachrizal Sinaga** — geographer / developer. A digital topography of the anthropocene: spatial analysis, cartography, and data visualization.

Live at [rizalsng.github.io](https://rizalsng.github.io)

## Features

- **Game of Life canvas background** — interactive (move your cursor to seed life), pauses on hidden tabs, static under `prefers-reduced-motion`
- **Map-sheet aesthetic** — viewfinder frame, marginalia with live coordinate readout (follows cursor, DMS format), scale bar, and a live WIB clock
- **Research logs** — latest Medium posts pulled via RSS, cached in localStorage as fallback
- **Project index** — hover/keyboard-focus previews with lerped parallax
- **Signal-decode intro** — title and metadata scramble in on load
- Film grain overlay, cyan selection color, SVG crosshair favicon

## Tech

Vanilla HTML/CSS/JS. No build step, no dependencies. Fonts: Cormorant Garamond + Space Mono.

## Local development

```sh
python3 -m http.server 8000
# open http://localhost:8000
```

## Deployment

GitHub Pages — push to `master` and it deploys automatically.
